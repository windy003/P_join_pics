# 图片合成器 - Image Composer (PyQt5版本)

一个简单易用的图片合成工具，可以导入多张图片，自由拖拽布置位置，并最终导出为一张完整的图片。

**使用PyQt5重写，完美支持高DPI显示器，导出原始分辨率图片！**

## 功能特性

✨ **新特性**
- 使用PyQt5框架，现代化界面设计
- **完美支持高DPI显示器**，界面清晰不模糊
- **导出原始分辨率图片**，保持最佳画质
- 智能缩放显示，编辑时流畅，导出时高清

🎨 **核心功能**
- 支持导入多张图片（PNG, JPG, JPEG, GIF, BMP格式）
- 通过鼠标拖拽自由移动图片位置
- **图片编辑功能**：
  - ✨ 缩放图片大小（Ctrl+= 放大，Ctrl+- 缩小）
  - ✨ Ctrl+滚轮缩放选中图片
  - ✨ Ctrl+0 重置大小
- 以原始分辨率显示所有图片（100%高清）
- 支持删除单个或多个选中的图片
- 一键清空整个画布
- 导出合成后的图片为PNG或JPG格式（保留所有编辑效果）
- 丰富的快捷键支持
- 画布缩放和视图调整功能

## 使用方法

### 启动程序

**PyQt5版本（推荐）：**
```bash
python image_composer_pyqt.py
```

**Tkinter版本（旧版）：**
```bash
python image_composer.py
```

### 快捷键

**文件操作**
- **Ctrl+O**: 导入图片（可以一次选择多张）
- **Ctrl+E** 或 **Ctrl+S**: 导出合成后的图片（原始分辨率）
- **Delete**: 删除当前选中的图片（可多选）

**图片编辑**
- **Ctrl+=** 或 **Ctrl++**: 放大选中的图片
- **Ctrl+-**: 缩小选中的图片
- **Ctrl+0**: 重置选中图片的大小
- **Ctrl+滚轮**: 缩放选中的图片

**视图控制**
- **Ctrl+P**: 适应窗口（自动调整视图显示所有图片）
- **鼠标滚轮**: 缩放视图
- **鼠标中键拖拽**: 平移画布

### 操作步骤

1. 启动程序后，按 `Ctrl+O` 或点击"导入图片"按钮
2. 在文件对话框中选择一张或多张图片（可以按住Ctrl键多选）
3. 图片会自动显示在画布上，每张图片会稍微错开位置
4. **编辑图片**：
   - 用鼠标点击并拖拽图片，调整位置
   - 选中图片后按 `Ctrl+=` 或 `Ctrl+-` 调整大小
   - 按住 `Ctrl` 并滚动鼠标滚轮也可以缩放图片
5. 可以点击图片选中后，按 `Delete` 键删除不需要的图片
6. 调整好所有图片位置和大小后，按 `Ctrl+E`（或 `Ctrl+S`）或点击"导出图片"按钮
7. 选择保存位置和文件名，完成导出（所有编辑效果会保留）

## 重要说明

### 关于图片清晰度

**PyQt5版本的优势：**
1. **界面显示**：完美支持高DPI显示器，界面清晰锐利
2. **原图显示**：界面上直接显示原始分辨率图片，100%高清无损
3. **所见即所得**：编辑时看到的效果就是导出的效果
4. **图片编辑**：支持自由缩放图片大小，所有编辑效果完美保留
5. **灵活布局**：可自由调整图片大小以适应布局需求

### 其他说明

- **所有图片以原始分辨率显示**，100%高清无损
- 如果图片太大，可以使用 `Ctrl+-` 缩小到合适的大小
- 导出时会自动计算所有图片占用的最小矩形区域，并添加适当边距
- 支持透明背景的PNG图片
- 导出PNG时保持透明度，导出JPG时使用白色背景
- **所见即所得**：编辑界面上看到的就是导出的效果
